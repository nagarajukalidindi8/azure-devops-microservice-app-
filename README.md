# Azure DevOps Microservice Project 

## Overview
This project demonstrates deployment of a simple Node.js microservice using Docker and Azure DevOps CI/CD pipeline.

## Features
- Containerized application using Docker
- Health check endpoint (/health)
- CI/CD pipeline integration
- Ready for Azure deployment

## Tech Stack
- Node.js
- Docker
- Azure DevOps
- GitHub

## How to Run
1. Clone repo
2. Run:
   docker build -t app .
   docker run -p 3000:3000 app
